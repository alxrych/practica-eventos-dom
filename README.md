## Galería de imágenes por categorías

Aplicación web interactiva que muestra imágenes organizadas por categorías. Permite al usuario explorar diferentes colecciones y hacer clic en miniaturas para intercambiarlas con la imagen principal.

### Funcionalidades
- **Visualización de categorías**: al cargar la página se muestran botones con todas las categorías disponibles.
- **Galería dinámica**: al seleccionar una categoría se despliega una imagen grande y varias miniaturas.
- **Intercambio de imágenes**: al hacer clic en una miniatura, esta pasa a ser la imagen principal y viceversa.
- **Diseño responsive**: adaptado a dispositivos móviles y de escritorio (Mobile First).

### Tecnologías
- HTML5 semántico
- CSS3 (Flexbox + Grid, variables, media queries)
- JavaScript (ES6+)
